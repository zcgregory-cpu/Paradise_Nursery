# Paradise Nursery

Paradise Nursery is a React and Redux shopping cart application for browsing houseplants, adding plants to a cart, changing quantities, and viewing order totals.

## Project Features

- Landing page with company information and a Get Started button.
- Product listing page with houseplants grouped into multiple categories.
- Redux-powered shopping cart with dynamic quantity totals.
- Cart page with item totals, quantity controls, delete buttons, checkout message, and continue shopping navigation.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
